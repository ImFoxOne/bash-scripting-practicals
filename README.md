# CIS 21031 - Platform Technologies
## Bash Scripting Practicals

This repository contains my bash scripting lab exercises for the Platform Technologies (CIS 21031) module at the South Eastern University of Sri Lanka.

### Student Details
* Registration Number: SEU-IS-21-ICT-034
* Academic Year: 2021/2022

### Lab Contents
* **Lab 01 & 02:** Basic Linux commands and environment setup
* **Lab 03:** Variables and arrays
* **Lab 04:** Arithmetic operations and basic conditions
* **Lab 05:** Numerical and string comparisons, file test operators
* **Lab 06:** Logical operators and case statements
* **Lab 07:** While and for loops
* **Lab 08:** Functions, parameter passing, and variable scope
* **Lab 09 & 10:** Comprehensive shell scripting exercises

### How to run the scripts
1. Clone this repository.
2. Navigate to the required lab folder.
3. Grant execute permissions to the script:
   `chmod +x script_name.sh`
4. Run the script:
   `./script_name.sh`
